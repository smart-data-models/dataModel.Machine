Description globale : **Cette entité contient une description harmonisée d'un modèle de machine générique. Cette entité est principalement associée au segment de l'industrie et aux applications IoT connexes. Le machineModel comprend une structure hiérarchique qui permet de regrouper les modèles de machines de manière flexible**.    
version : 0.0.1
